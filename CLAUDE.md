## What This Is

GreenLight is a personal financial planning SPA for tracking investments, projecting cash flow, planning home/car purchases, and determining financial readiness. React 19 + Vite, no TypeScript. All state lives in localStorage (key: `"greenlight"`) with IndexedDB (via Dexie) for time-series caching of prices and rates.

## Architecture

**Pure calculation layer** (`src/lib/`): All financial math is pure functions with no side effects — tax engine, mortgage amortization, purchase planning, readiness projections. These are independently testable and the tests live in `src/lib/__tests__/`.

**Data flow**: `useStorage()` hook loads/saves localStorage state → pure calculation functions compute derived values → `usePrices()` fetches live prices (Yahoo Finance, CoinGecko, Gemini) with IndexedDB caching → components render results.

**State schema** (`src/data/defaults.js`): Versioned via `SCHEMA_VERSION` with automatic migration in `src/lib/storage.js`. Adding new state fields requires bumping the schema version and handling migration from previous versions.

**Styling**: CSS-in-JS via inline styles. All design tokens (colors, fonts, shared styles) are in `src/theme.js` — the "Mission Control" dark theme. Components import from theme.js, not CSS files.

**API proxies**: Dev server proxies `/api/yahoo`, `/api/coingecko`, `/api/finnhub`, `/api/fred` to external APIs (configured in `vite.config.js`). Production uses Nginx reverse proxies (see Dockerfile).

**Generated data**: `src/data/conformingLimits.js` is generated by `scripts/buildLoanLimits.mjs` from FHFA source data in `data/fhfa/`. Do not edit this file directly — run `npm run build:limits` instead.

## Key Conventions

- Pages go in `src/pages/`, reusable components in `src/components/`, business logic in `src/lib/`, static data in `src/data/`, hooks in `src/hooks/`
- Calculation functions must remain pure — no DOM access, no fetch calls, no side effects
- Tests focus on the calculation layer; test files mirror the source file they test (e.g., `taxEngine.js` → `__tests__/taxEngine.test.js`)
- Routes are defined in `src/App.jsx`; sidebar navigation is in `src/components/Layout.jsx`
- ESLint flat config (v9+) with react-hooks and react-refresh plugins
